## Descripción

Repositorio del proyecto de clase desarrollado durante las sesiones prácticas de la asignatura **Desarrollo Backend**.

El proyecto se construye progresivamente en clase mediante la implementación de diferentes componentes del framework **Laravel**, permitiendo aplicar de forma práctica conceptos fundamentales del desarrollo backend.

A lo largo del curso se desarrollan funcionalidades como:

- Configuración inicial del entorno de desarrollo con Laravel.
- Definición de rutas y control de navegación mediante `web.php`.
- Creación y organización de vistas utilizando **Blade**.
- Implementación de estilos personalizados mediante **CSS**.
- Estructuración de una aplicación web básica siguiendo las convenciones del framework.
- Control de versiones del proyecto mediante **Git**.

Este repositorio refleja el avance progresivo del trabajo realizado en cada sesión de clase.

## Requisitos del Sistema

- PHP 8.2 o superior
- Composer
- Git
- Servidor local (XAMPP o servidor integrado de Laravel)

---

## Instalación

git clone https://github.com/Kmilap/proyecto-clase-backend.git  
cd mi-perfil-laravel  
composer install  
cp .env.example .env  
php artisan key:generate  

---

## Ejecución

php artisan serve  

Abrir en el navegador:

http://127.0.0.1:8000
